# Agent Loop Runner

CLI utilities for agent automation loops (PR babysitting via `@cursor/sdk`).

## Use

```bash
cd ~/Projects/99_System/agent-loop-runner
npm install
npm run babysit-pr
```

## Scripts

- `npm run babysit-pr` — PR triage loop entrypoint
- `npm run build` / `npm run typecheck` — TypeScript compile checks
